# gem uninstall all

## Installation

```
$ go get github.com/naoty/gem-uninstall-all
```

## Usage

```
$ gem-uninstall-all
```

### ext

This command is compliant with the interface provided by [naoty/ext](https://github.com/naoty/ext).

```
$ go get github.com/naoty/ext
$ alias gem="ext gem"
$ gem uninstall all
```
